# recur-date-based-docs

Extended documentation site for the [recur-date-based](https://github.com/NavasardianMichael/recur-date-based) package. Provides comprehensive API reference, popular examples, format tokens, and usage guides.

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm (this project uses pnpm only)

### Install

```bash
pnpm install
```

### Development

```bash
pnpm start
```

Runs the app at [http://localhost:3000](http://localhost:3000).

### Build

```bash
pnpm run build
```

### Deploy

```bash
pnpm run deploy
```

Deploys to GitHub Pages (configured via `homepage` in `package.json`).

## Scripts

| Command              | Description              |
| -------------------- | ------------------------ |
| `pnpm start`         | Start development server |
| `pnpm run build`     | Build for production     |
| `pnpm run deploy`     | Deploy to GitHub Pages   |
